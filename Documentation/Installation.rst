============
Installation
============

..  _installation-composer:

Install T3 backend login tweaks with Composer
=======================================

Install the extension via Composer:

..  code-block:: bash

    composer req ideative/t3-be-login

See also `Installing extensions, TYPO3 Getting started <https://docs.typo3.org/permalink/t3start:installing-extensions>`_.

After installation:

1. Activate the extension in TYPO3 if needed.
2. Open **System > Login appearance**.
3. Save your preferred defaults.

..  _installation-classic:

Install T3 backend login tweaks in Classic Mode
=========================================

Or download the extension from `https://extensions.typo3.org/package/id_be_login <https://extensions.typo3.org/package/id_be_login>`_ and install it in
the Extension Manager.
